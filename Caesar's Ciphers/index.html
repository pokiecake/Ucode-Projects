Caesar's box encoder and decoder

How Caesar box works:

  1. Think of a message you want to encode.
    i.e. abcdefghi
  2. Take the length of the message, then take the square root of it (Caesar's box won't work if the length isn't a perfect square).
    i.e. abcdefghi = 9 letters. square root of 9 = 3
  3. Now, write the letters into rows and columns. Write the letters from left to right until the number of columns is equal to the square root of the length. When this happens, move on to the next row and repeat until all letters are used. The result should look like a square.
    i.e.
    a b c
    d e f
    g h i
  4. Now, write the letters from the first column down, then moving to the next column until all letters are written. Now, your message has been encoded.
    i.e. adgbehcfi
  5. To decode, repeat the same steps.
